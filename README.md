# AWS Certified Solutions Architect - Associate (SAA-C03) Study Hub

This repository serves as a comprehensive guide and resource center for anyone preparing for the AWS SAA-C03 exam. I developed this roadmap while balancing a full-time professional role at Rysun Labs, proving that with disciplined planning and dedication, clearing this exam is possible even with a hectic schedule.

## Overview
This guide focuses on a 4-6 week preparation window, emphasizing core architectural principles, hands-on practice within the AWS Free Tier, and rigorous exam simulation.

## 6-Week Study Roadmap

### Phase 1: Knowledge Acquisition (Weeks 1-3)
* **Primary Resource:** [Ultimate AWS Certified Solutions Architect Associate - Stephane Maarek](https://www.udemy.com/course/aws-certified-solutions-architect-associate-saa-c03/)
* **Action:** Complete the video course. Follow along with the hands-on sections.
* **Focus:** Core services (VPC, EC2, S3, RDS, IAM) and the AWS Global Infrastructure.

### Phase 2: Architectural Deep Dive (Week 4)
* **Primary Resource:** [Official AWS Exam Topics & Whitepapers](https://docs.aws.amazon.com/pdfs/aws-certification/latest/solutions-architect-associate-03/solutions-architect-associate-03.pdf#solutions-architect-associate-03)
* **Action:** Study the **AWS Well-Architected Framework** (6 Pillars).
* **Focus:** Reliability, Security, and Cost Optimization.

### Phase 3: Practice & Refinement (Weeks 5-6)
* **Primary Resource:** [Tutorials Dojo Practice Exams](https://portal.tutorialsdojo.com/courses/aws-certified-solutions-architect-associate-practice-exams/)
* **Action:** Take full-length exams in "Timed Mode." 
* **Focus:** Understanding "why" the correct answer is right and "why" the others are wrong.
